PCMAD(1)
========
:doctype: manpage


NAME
----
pcmad - Page Cache My Assets daemon


SYNOPSIS
--------
*pcmad* [-e 'ENDPOINT']


DESCRIPTION
-----------
The pcmad(1) daemon listens for requests and executes them.
Requests are documented in +pcma(5)+.

OPTIONS
-------
*-e* 'ENDPOINT':
  Specify the endpoint to bind to. Defaults to +ipc:///var/run/pcma.socket+.
